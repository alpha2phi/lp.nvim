# lp.nvim

Generate output for embedded code snippets.

lp.nvim is a plugin to generate code output for embedded code snippets in Markdown
document

Currently support:

- JavaScript
- Python

Available commands:

- LpRun <lang>
- LpClean
  
### Languages

- [x] Python
- [x] JavaScript
- [ ] Rust
- [ ] Go
- [ ] TypeScript
- [ ] Lua
- [ ] Bash
- [ ] ZSH
